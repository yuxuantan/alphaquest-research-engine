# spy_1d_volume_attention_continuation_1530

Two-sided ES continuation using prior SPY 1-day return direction and 20-day relative-volume attention rank. The signal uses prior-day SPY features only, then waits for completed ES price movement and signed aggregate orderflow before next-bar entry.

Pre-PnL density at the fixed review settings was 855 full-sample signals, about 55.41/year, and 54.59/year in the limited-core reference window.
