# spy_5d_volume_attention_continuation_1530

Two-sided ES continuation using prior SPY 5-day return direction and 63-day relative-volume attention rank. The signal uses prior-day SPY features only, then waits for completed ES price movement and signed aggregate orderflow before next-bar entry.

Pre-PnL density at the fixed review settings was 882 full-sample signals, about 57.16/year, and 58.49/year in the limited-core reference window.
