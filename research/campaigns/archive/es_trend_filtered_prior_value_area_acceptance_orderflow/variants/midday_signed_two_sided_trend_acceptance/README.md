# midday_signed_two_sided_trend_acceptance

Two-sided ES prior-value-area acceptance variant. It trades 11:00-14:00 ET when a completed 5-minute bar accepts beyond prior VAH or VAL, broad signed-volume imbalance confirms direction, and slower completed trend windows agree. Entry is next bar open; stop is percent from entry; target is fixed-R; unresolved trades flatten at 15:55 ET.
