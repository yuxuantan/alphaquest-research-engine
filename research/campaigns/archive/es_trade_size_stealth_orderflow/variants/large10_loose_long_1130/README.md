# large10_loose_long_1130

This variant follows positive completed 30-minute large10 signed-flow imbalance
at 11:30 ET when larger-trade pressure materially exceeds residual smaller-flow
imbalance. It enters on the next 1-minute bar open and flattens by 12:30 ET.

The mechanic uses the broader large10 bucket to test whether medium/large flow
contains more information than residual smaller flow.
