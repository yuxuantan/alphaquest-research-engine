# morning_signed_vah_trend_acceptance_long

Long-only ES prior-value-area acceptance variant. It trades only when a completed 5-minute morning bar closes above prior RTH VAH, signed-volume imbalance confirms buyer participation, and two completed-bar trend windows show higher-high/higher-low structure. Entry is next bar open; stop is percent from entry; target is fixed-R; unresolved trades flatten at 15:55 ET.
