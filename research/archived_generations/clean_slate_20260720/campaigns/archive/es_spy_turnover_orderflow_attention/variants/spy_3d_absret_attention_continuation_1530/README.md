# spy_3d_absret_attention_continuation_1530

Two-sided ES continuation using prior SPY 3-day return direction and three-day absolute-return-times-volume attention rank. The signal uses prior-day SPY features only, then waits for completed ES price movement and signed aggregate orderflow before next-bar entry.

Pre-PnL density at the fixed review settings was 849 full-sample signals, about 55.02/year, and 55.89/year in the limited-core reference window.
