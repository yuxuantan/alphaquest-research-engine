# full_session_two_sided_reclaim

This variant expresses the prior-day stop-run reclaim edge across the full RTH
session. From 09:35 through 15:30 ET, it fades completed 5-minute sweeps of the
prior RTH low that close back above the level and completed sweeps of the prior
RTH high that close back below the level.

Tunable parameters are fixed before testing: reclaim window, minimum volume
ratio, percent stop, and fixed-R target.
