# all_available_1515_signed120_continue

At 15:15:00 ET, trade NQ continuation in the completed open-to-signal move direction on all available SPX 0DTE sessions, excluding standard monthly expirations, only when completed 120-minute NQ signed flow aligns with the move.

Declared before PnL testing. Entry grid: min_abs_move_ticks [100, 140, 180], min_orderflow_imbalance [0.0, 0.0025, 0.005]. Stop grid: [0.004, 0.006, 0.008]. Target grid: [1.0, 1.5, 2.0].
