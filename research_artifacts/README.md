# Durable Research Artifacts

This directory contains durable audits and research decisions. Historical flat paths remain stable because campaign definitions and immutable run evidence reference them.

Use `views/artifacts/` and the artifact catalog for categorized navigation. New repository-wide outputs should use these logical classes:

- `catalogs/`
- `audits/repository/`
- `audits/methodology/`
- `audits/density/`
- `campaigns/<campaign_id>/`
- `qualification/`
- `search_gates/`
- `cleanup/`

Do not relocate historical evidence without a reviewed migration map and reference-integrity audit.
