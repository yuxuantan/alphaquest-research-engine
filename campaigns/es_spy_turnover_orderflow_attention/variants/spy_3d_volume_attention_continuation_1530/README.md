# spy_3d_volume_attention_continuation_1530

Two-sided ES continuation using prior SPY 3-day return direction and 63-day relative-volume attention rank. The signal uses prior-day SPY features only, then waits for completed ES price movement and signed aggregate orderflow before next-bar entry.

Pre-PnL density at the fixed review settings was 877 full-sample signals, about 56.84/year, and 54.59/year in the limited-core reference window.
