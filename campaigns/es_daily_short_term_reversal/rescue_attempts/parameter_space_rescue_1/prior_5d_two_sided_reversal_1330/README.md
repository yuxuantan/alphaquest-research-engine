# prior_5d_two_sided_reversal_1330

At the 13:30 ET completed 5-minute bar, fade the completed five-session RTH close-to-close move: long after a sufficiently negative move and short after a sufficiently positive move.

This variant tests whether slower short-term pressure over roughly one trading week reverses intraday without using current-session future information.

Rescue attempt 1 keeps the same two-sided direction mode, lookback, signal time, modules, data, costs, and gates. It changes only the declared threshold, stop, and target parameter space plus matching fixed defaults.
