# afternoon_two_sided_reclaim

This variant expresses the same prior-day stop-run reclaim edge during the
afternoon window. From 13:30 through 15:30 ET, it fades completed 5-minute
sweeps of either prior RTH extreme after the signal bar closes back inside the
level.

Tunable parameters are fixed before testing: reclaim window, minimum volume
ratio, percent stop, and fixed-R target.
