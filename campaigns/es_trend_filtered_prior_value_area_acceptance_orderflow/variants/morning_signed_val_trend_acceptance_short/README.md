# morning_signed_val_trend_acceptance_short

Short-only ES prior-value-area acceptance variant. It trades only when a completed 5-minute morning bar closes below prior RTH VAL, signed-volume imbalance confirms seller participation, and two completed-bar trend windows show lower-high/lower-low structure. Entry is next bar open; stop is percent from entry; target is fixed-R; unresolved trades flatten at 15:55 ET.
