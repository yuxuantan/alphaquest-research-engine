# morning_1030_slot_persistence

At 10:30 ET, trade the 10:30-11:00 slot in the same direction as the prior
same-slot rolling mean return when its absolute value clears the configured bps
threshold. The signal uses only prior sessions and exits no later than 11:00 ET.
