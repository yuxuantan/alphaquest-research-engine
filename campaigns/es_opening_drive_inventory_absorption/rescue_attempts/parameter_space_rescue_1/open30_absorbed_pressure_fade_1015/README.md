# open30_absorbed_pressure_fade_1015

This variant shorts after a completed first-30-minute upside opening drive if the drive used high-volume buy pressure but price has not extended cleanly by 10:15 ET. The signal is evaluated at the 10:14 close for intended next-bar execution at 10:15 ET and flattens at 10:45 ET.

Tunable parameters are fixed before testing: opening return threshold, opening volume-rank threshold, percent stop, and fixed-R target.
