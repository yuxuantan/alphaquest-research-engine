# open30_flow_continuation_1030

This variant goes long after a completed first-30-minute RTH opening drive when price return, signed trade imbalance, cumulative delta, and volume-rank context remain aligned. The signal is evaluated at the 10:29 close for intended next-bar execution at 10:30 ET and flattens at 12:00 ET.

Tunable parameters are fixed before testing: opening return threshold, opening volume-rank threshold, percent stop, and fixed-R target.
