# open30_flow_continuation_1030

Use only the completed 09:30-09:59 ET opening window. At 10:30 ET, go long when opening return, signed-flow imbalance, prior-session opening-volume rank, and current cumulative delta remain aligned; flatten at 12:00 ET unless stop or target is hit.

Tunable parameters are fixed before testing: opening return threshold, opening volume-rank threshold, percent stop, and fixed-R target. NQ opening-return thresholds were chosen from unconditional opening-window distributions before staged PnL testing.
